API Pays Européens

Pour ce projet, j'ai utilisé des API pour la réalisation de ce projet

Ce site permet :

  • Afficher une liste de pays européen : les utilisateurs peuvent explorer une liste de pays.
  • Afficher les détails d'un pays : permettre aux utilisateurs d'en savoir plus sur le pays.
  • Langages : HTML et JavaScript.
    API : REST Country, OpenWeather, Wikipedia.

La réalisation de ce projet m'a permis de renforcer mes compétences en HTML mais surtout en JavaScript. Cela me montre qu'on peut réaliser des sites internet sans initier nous même une base de données avec d'innombrables données. Ce projet a enrichi mes connaissances en interaction backend.
